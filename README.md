# Credit Decision Simulation Package

A comprehensive system for simulating credit decision processes with multiple scores, stages, and trade-off analysis capabilities.

## Features

- **Multiple Score Evaluation**: Compare different credit scoring models
- **Flexible Simulation Stages**: Configure credit approval, anti-fraud checks, and conversion stages
- **Trade-off Analysis**: Analyze the relationship between approval rates and default rates
- **Optimization**: Find optimal cutoffs based on different business objectives
- **Visualization**: Create insightful visualizations of trade-offs and relationships
